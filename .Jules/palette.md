# Palette's Journal

## 2024-05-21 - [Explaining Auto-scroll Behavior]
**Learning:** In live-streaming log viewers, "Auto-scroll" can be confusing if it turns off automatically when a user scrolls up. Users might think it's a bug if they don't realize that manual scrolling intentionally pauses the auto-follow behavior.
**Action:** Always provide a tooltip or clear visual indicator that explains why auto-scroll might have paused and how to re-enable it.
